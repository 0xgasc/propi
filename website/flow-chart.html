PROPERTY OWNER UPLOADS DOCUMENTS

REQUIRED DOCUMENTS:
  • • Título de Propiedad (Property Title)
  • • RGP Certification
  • • Property Photos
  • • Municipal Permits
WHAT HAPPENS:

3

OWNER CHOOSES TOKENIZATION TYPE

EL PROPIETARIO ELIGE EL TIPO DE TOKENIZACIÓN

W
WHOLE PROPERTY NFT
  • • Single NFT = Complete ownership
  • • Traditional sale to one buyer
  • • No fractionalization
  • • Suitable for high-value properties
Example: $450K house → 1 NFT
F
FRACTIONAL SHARES
  • • Property split into shares
  • • Multiple investors can participate
  • • Lower investment minimums
  • • Revenue sharing enabled
Example: $75K house → 300 shares @ $250 each
FRACTIONAL PARAMETERS (OWNER SETS):
Total Shares:
100, 1000, or 10000
Price per Share:
Auto-calculated from property value
Minimum Investment:
Owner-defined (e.g., 4 shares = $1000)
4

SMART CONTRACT EXECUTION

EJECUCIÓN DE CONTRATO INTELIGENTE

PROPERTYREGISTRY.SOL:
  • • Mints Property NFT to owner
  • • Stores metadata URI (permanent link)
  • • Records property details on-chain
  • • Enables secure transfers
FRACTIONALPROPERTY.SOL:
  • • Creates fractional shares (ERC-1155)
  • • Sets up investment parameters
  • • Enables revenue distribution
  • • Manages investor registry
BC
BLOCKCHAIN RESULT:

Property NFT: Unique token representing ownership
Token URI: Points to permanent metadata
L2 Network: Low-cost, fast transactions (~$0.01)
