How Propius Works

From property deed to blockchain token in 6 simple steps

1

Upload Property Documents

Required Documents

  • • Título de Propiedad (Property Title)
  • • RGP Certification
  • • Property photos
  • • Municipal permits

What Happens

  • • Documents stored on Arweave (permanent)
  • • Property data extracted automatically
  • • Unique property ID generated
  • • Document integrity verified

Result: Your documents get a permanent Arweave URL and SHA-256 hash for verification.

2

Notary Verification

Verification Process

  • • Document authenticity check
  • • RGP registry cross-reference
  • • Ownership confirmation
  • • Legal compliance review

Digital Signature

  • • Notary signs with digital certificate
  • • Signature stored on blockchain
  • • Timestamp recorded permanently
  • • Property marked as verified

Result: Property is marked as VERIFIED and ready for tokenization.

3

Choose Tokenization Type

Whole Property NFT

  • • Single NFT = Complete ownership
  • • Traditional sale to one buyer
  • • No fractionalization
  • • Best for high-value properties
Example: $450K house → 1 NFT

Fractional Shares

  • • Property split into shares
  • • Multiple investors participate
  • • Lower investment minimums
  • • Revenue sharing enabled
Example: $75K house → 300 shares @ $250 each

Your Choice: Set total shares, price per share, and minimum investment amount.

4

Smart Contract Deployment

PropertyRegistry Contract

  • • Mints Property NFT to your wallet
  • • Links to Arweave document storage
  • • Records property details on-chain
  • • Enables secure transfers

FractionalProperty Contract

  • • Creates fractional shares (if chosen)
  • • Sets investment parameters
  • • Enables revenue distribution
  • • Manages investor registry

Result: Your property is now tokenized on Base blockchain with ~$0.01 transaction costs.

5

Investment & KYC

Tier 1: $500-$5K

  • • Basic ID verification
  • • Address confirmation
  • • Perfect for families

Tier 2: $5K-$50K

  • • Enhanced KYC process
  • • Income verification
  • • Source of funds docs

Tier 3: $50K+

  • • Accredited investor status
  • • Net worth documentation
  • • Professional verification

Process: KYC verification → USDC payment → Tokens to wallet → Revenue sharing begins.

For Investors

Automated: Monthly rental income and governance voting handled by smart contracts.
